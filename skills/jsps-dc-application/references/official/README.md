# 官方资料(链接,未随仓打包 / Official documents — linked, not bundled)

JSPS 官方文档版权归 JSPS 所有,本公开仓**不二传 PDF**,请从官方下载当年度版本。
The official JSPS documents are not redistributed here; download the current year's
versions from JSPS directly.

> 本 skill 起草时基于 **令和9(2027)年度採用分**。**申请别的年度务必换成当年様式**(様式逐年微调)。
> Built against the **FY2027 (令和9)** cycle. For other years, fetch that year's forms.
