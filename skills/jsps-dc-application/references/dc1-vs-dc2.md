# DC1 vs DC2 差异 + 日/英措辞规范

## 区分差异(様式同,侧重不同)

| | DC1 | DC2 |
|---|---|---|
| 资格 | 申请时博一在学/将入学(例:2026年4月入・進学 見込) | 博二/博三(例:2025年4月已入・進学) |
| 研究期间 | 最大 **3 年** | 最大 **2 年** |
| 现指导教员(=评价书作成者) | 硕士课程指导教员(或现职上司/直近指导者) | 博士课程现指导教员 |
| 奨励費 A区分上限 | 3年 240万 / 2年 160万 / 1年 80万 | 2年 160万 / 1年 80万 |
| **写作侧重** | 业绩通常较薄 → **重构想的独创性、研究潜力、计划说服力**;着想に至った経緯讲透;用硕士期成果+预备结果证潜力 | 已有进展 → **重既取得成果、可行性、主体性**;【４】自己分析业绩更充实;预备实验"已着手·已观察到差异"强化可行 |
| 自己分析(节4)落点 | "我**会成为**优秀研究者"——成长性、学习力、跨域吸收力 | "我**正在成为**优秀研究者"——已独立产出、主导能力 |

> 两者都共用同一份 DC 様式与同一套 審査の観点(①背景・着想・方法独創・展望 / ②研究者将来性)。差异体现在**证据强度与叙事重心**,不在结构。

## 日语写作规范(である調)

- 全文 **である調**(学术体),非です・ます調。例:「~と考えられる」「~を明らかにする」「本研究の独創性は…にある」。
- 第一人称用 **「申請者」**(非「私」「我々」;单独 fellowship 不用「我々」除非指共同研究团队客观存在)。
- 标点:句读用「、。」;强调用**太字**与「」;列举用 ①②③ 或 ・。
- 漢字/カナ:课题名避免化学式·数式(用漢字カナ;DNA 等惯用字母可);JIS第1·2水準外字符与丸数字①②勿用于 Web 输入的研究課題名(但申請内容ファイル正文中作小标题项番 ①②③ 可)。
- 年号:正文西历即可;申請書情報栏位按系统要求。

## 英语写作规范

- DC 申請内容ファイル可全英文(作成要領明确「日本語又は英語」)。
- 学术正式体,主动语态优先;第一人称用 **"the applicant"**(对应「申請者」),避免随意的 "I/we"(除非客观共同研究)。
- 术语首次出现给全称+缩写;图表标签可英文,但若投日本审查セット,关键标签建议中立/英文并确保灰度可辨。
- 篇幅:英文同样受 1/2/1/2 页限制——英文通常更占空间,注意精简。
- 参考英文范例的写法(如 Steve Heim 的 JSPS DC 英文申请经验,见 reference.md §9)。

## 提醒
- 不论日/英,**全文语言统一**,勿日英混杂(术语括注除外)。
- 评价书由现指导教员写(DC=1名),申请人不可见;起草时可提醒用户**尽早联系导师**并提供研究概要,便于其写出契合的评价书。
